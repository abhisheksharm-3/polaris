# Polaris Writing Standard

<!-- docs, and marketing copy. No exceptions. Injected every session. -->

Every sentence must pass this filter before it goes out. These patterns are the clearest signals
that text was generated by an LLM, and they erode trust.

## Scope

This standard governs every line of natural language Polaris emits: code comments, commit
messages, PR titles and bodies, documentation, and marketing copy. A commit message is held to
the same bar as an article.

## Banned vocabulary

Never use these words. Find a concrete, specific alternative.

| Banned | Replace with |
|---|---|
| delve | explore, examine, look at, dig into |
| tapestry | a specific description of the actual thing |
| pivotal | critical (if true), key, or cut it |
| underscore (as a verb) | show, reveal, confirm, mean |
| testament | proof, sign, evidence |
| vibrant | specific: busy, active, high-growth |
| showcase | demonstrate, display, or cut |
| seamless / seamlessly | cut it, or describe what actually happens |
| leverage | use |
| intricate | describe the actual complexity |
| robust | strong, reliable, or describe what it does |
| nestled | cut it |
| boasts | has |
| groundbreaking | name the actual innovation |
| fostering / cultivating | building, creating, growing |
| holistic | say what it covers |
| synergy | cut it |
| cutting-edge | name the actual technology |

Sentence openers to avoid: `Additionally,` `Furthermore,` `Moreover,` at the start of a sentence.

## Banned sentence structures

- **"Not only X but also Y"** and **"Not just X, but Y"**. Signals hedging or false completeness.
  State the two facts plainly.
- **The importance preamble.** "It is important to note that..." / "It is worth noting that...".
  Start with the fact instead.
- **Rule-of-three padding.** "efficient, scalable, and maintainable." Use three items only when
  there are exactly three that matter; otherwise say the one that matters most.
- **"Serves as" / "stands as" / "acts as" instead of "is".** Use "is". It is fine.
- **The challenge-and-future closer.** Do not end with vague challenges followed by unprompted
  optimism about the future.

## Banned analytical moves

- **Vague attribution.** "Experts argue", "observers note", "many companies find". Name the
  source, link it, or say what you know from doing the work.
- **Significance inflation.** "This represents a broader shift..." without data. Prove it or cut
  it. No "-ing" tails that claim importance ("...highlighting its significance").
- **Unsupported debate.** "This generated considerable debate" without citing the debate.
- **Elegant variation.** Rotating five synonyms for one concept to avoid repetition. Repeat the
  right word. Consistency beats variety.

## Forbidden formatting habits

| Pattern | Rule |
|---|---|
| Title Case In Every Heading | Use sentence case |
| Bold on every term in a list | Bold only genuinely critical terms |
| Em dashes everywhere — for every — pause | Use commas and periods |
| Thematic breaks before every section | Only where logically needed |
| Skipping heading levels (H2 to H4) | Keep the hierarchy |

## The positive rule

Every paragraph contains at least one thing specific enough to be wrong: a number, a named
product, a named company, a concrete outcome. A paragraph vague enough to apply to any topic
gets cut.

## How to write like a human

1. Say the thing directly. "The API is slow", not "The API faces performance challenges".
2. Use "is" without embarrassment.
3. Earn significance; do not claim it.
4. Let sentences end. Drop trailing qualifiers like "...which is crucial for modern applications".
5. One qualifier per claim. Not "it is generally considered that it may potentially...".
6. Vary sentence length. Short. Then medium. Then a longer one when the idea needs room.
7. No unsolicited predictions about the future unless asked.

## Quick self-check before submitting

- Any word from the banned list
- A sentence opening with "Additionally," "Furthermore," "Moreover,"
- A three-item list padded to look thorough
- A "challenges then future outlook" closer
- "serves as" / "stands as" instead of "is"
- Significance claims without evidence
- Weasel attribution
- Uniform sentence length throughout
